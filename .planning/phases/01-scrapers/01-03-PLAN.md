  <how-to-verify>
1. Open the politsei.ee daily crime report listing: https://www.politsei.ee/et/uudised/oeoepaeevainfo

2. Open at least 10 articles spanning different months (aim for 2-3 articles per month across 4-5 different months). Good starting points from research: Aug 2024, Mar 2024, Jan 2025, Oct 2024, Jun 2024.

3. In each article, search for the word "euro" (Ctrl+F). For every sentence containing "euro", record:
   - The EXACT sentence text (copy-paste, preserve spaces and diacritics)
   - The EUR amount that sentence states
   - What pattern type it is

4. Specifically look for:
   - "Kahju on X eurot" / "Kahju on üle X euro"
   - "peteti välja X eurot"
   - "Pettusega tekitatud kahju on X eurot"
   - "summas X eurot"
   - "kantud … X eurot"
   - Any other "euro" sentence not matching the above
   - Articles with NO euro amount (note these too — expected to be common)

5. Note the number format: does "70 000" use a space, period, or comma as thousands separator?

6. Create .planning/phases/01-scrapers/01-DISCOVERY-PASS.md with the following sections:
   - **Articles Read** (URLs and dates, at least 10)
   - **All Observed EUR Sentences** (exact text, grouped by pattern type)
   - **Confirmed Regex Patterns** (one regex per pattern, tested mentally against your examples)
   - **Edge Cases** (any unusual formatting, zero-amount articles, "üle" approximations)
   - **Patterns NOT seen in research** (note if you found new variants)
   - **Recommended EUR_PATTERNS array** (ready-to-copy TypeScript)

7. Commit the file: `git add .planning/phases/01-scrapers/01-DISCOVERY-PASS.md && git commit -m "docs(01-scrapers): police article discovery pass"`
